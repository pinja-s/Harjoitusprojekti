# Harjoitusprojekti
Malli-repositorio ohjelmoinnin kurssin harjoitustyötä varten. Repositorioon on tehty valmiiksi 4 kehityshaaraa, branchia, jotta jokainen ryhmän jäsen voi tehdä kehitystyötä omassa haarassaan. Näiden haarojen sisältö yhdistetään *paivitys*-haaraan, joka projektin valmistuessa yhdistetään *main*-haaraan. **HUOM! Main-haaraan yhdistetään sisältö keskitetystä kehityshaarasta, mutta main-haarassa ei tehdä muokkauksia!**
